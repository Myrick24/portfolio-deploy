Harvest: E-commerce Application

Harvest is a comprehensive e-commerce platform I developed to provide businesses with a modern online shopping solution. The application features a responsive product catalog, shopping cart functionality, secure checkout process, and integrated payment gateway.

Key features include inventory management, customer accounts, order tracking, promotional tools, and detailed analytics. The frontend was built with React for a smooth user experience, while the backend uses Node.js with Express to handle data processing and API endpoints.

This project required careful attention to user experience, security, and performance optimization to ensure fast loading times even with large product catalogs.

Security Ni Jomar: Authentication System

Security Ni Jomar is a robust authentication and security system I developed to provide enhanced protection for web applications. The core functionality includes a secure login system with multi-factor authentication, comprehensive email verification process, and intelligent session management with automatic logout for inactive users.

The system implements industry best practices including password hashing with bcrypt, JWT for secure token-based authentication, rate limiting to prevent brute force attacks, and detailed security logs for monitoring and audit purposes.

This project demonstrates my ability to implement critical security features that protect user data while maintaining a smooth, frustration-free user experience.
